## ② North Star Link（北極星との接続）

```
Claim成立
  → 真の3bモデルでも多様性がCFRを低下させる
  → 「多様性がshared_premisesの代替として機能する」

    普遍的な原理である可能性が示される
```

**接続の論理:**
```
Claim成立
  → CFR(3b異種) < CFR(3b同種)（真の3bで確認）
  → 多様性 = 異なる判断基準による疑似shared_premises
```

## CFRとグループシンクの関係（学習チャット指摘への対応）

```
学習チャットの指摘:
  「CFRと全員一致率の間にトートロジーの可能性」

確認すべき数式:
  CFR = wrong_unanimous / total_unanimous

  全員一致率が下がると:
  total_unanimousが減る → CFRの分母が減る

  ただしwrong_unanimousも減る場合:
  CFRの値は変わらない可能性がある

今回の観察（MVE-03）:
  3b同種: unanimous=96, wrong_unanimous=37 → CFR=38.5%
  3b異種: unanimous=47, wrong_unanimous=16 → CFR=34.0%

  全員一致が半減（96→47）
  wrong_unanimousも半分以下（37→16）
  CFRは微減（38.5%→34.0%）

解釈:
  「全員一致する時の誤り率」はほぼ変わっていない
  多様性の主な効果は「全員一致を防ぐこと」
  「全員一致した時の質」は変わらない

今回のMVEでも同じ数字を報告して
トートロジーの可能性を明示する
```
