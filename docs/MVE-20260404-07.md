## flip rateの計算方法

```python
# 各タスクのflip rate計算
def calc_flip_rate(task_results: list[dict]) -> float:
    """
    task_results: 同一タスクの5試行分の結果
    各要素: {"decision": "CORRECT"/"INCORRECT", ...}
    """
    decisions = [r["decision"] for r in task_results]
    p_correct = decisions.count("CORRECT") / len(decisions)
    p_incorrect = 1 - p_correct
    flip_rate = 1 - max(p_correct, p_incorrect)
    return flip_rate

# 解釈:
# p_correct=5/5=1.0 → flip_rate=0.0（全試行一致・決定論的）
# p_correct=4/5=0.8 → flip_rate=0.2（1回だけ異なる）
# p_correct=3/5=0.6 → flip_rate=0.4（2回異なる）
# p_correct=2/5=0.4 → flip_rate=0.4（最大のぶれ）
# p_correct=1/5=0.2 → flip_rate=0.2
# p_correct=0/5=0.0 → flip_rate=0.0（全試行一致・ただし全誤り）

# タスクセット全体のflip rate
# = mean(flip_rate) over all tasks
```
