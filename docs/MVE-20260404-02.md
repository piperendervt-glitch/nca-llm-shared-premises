## ① Feature Claim（機能の主張）

```
「NCA構造におけるグループシンクの発生率は
 モデルサイズに依存する：
 qwen2.5:3b × 3ノードでは高いCFRを示すが
 qwen2.5:7b × 3ノードでは低いCFRにとどまる」
```

**Claimが曖昧でないか確認:**
- [x] 主語（何が）が明確か
      → 「qwen2.5:3b × 3ノード vs qwen2.5:7b × 3ノード」
- [x] 述語（どう変わるか）が測定可能か
      → CFR（wrong_unanimous / total_unanimous）で測定
- [x] 対象（どのデータ・条件で）が特定されているか
      → world_consistencyタスク・100問
      → モデルシリーズをqwen2.5に統一
      → NCA構造（steps=3・同種3ノード）

## ② North Star Link（北極星との接続）

```
Claim成立
  → モデル能力が低い（3b）ほど
    他ノードの出力に引きずられやすい
  → 能力が低い = shared_premisesを
    自分で保持できない
  → shared_premisesを持てないエージェントが
    集合すると集団的な誤りが増える
  → 北極星A-4
    「shared_premisesの欠如が失敗の根本」
    を実験的に示す証拠になる
```

**接続の論理:**
```
Claim成立
  → CFR(3b) >> CFR(7b) が統計的に確認される
  → 「モデル能力とshared_premises保持能力は相関する」
  → 「能力の低いエージェントほど
     shared_premisesなしで誤った合意をする」
  → 北極星A-4の「shared_premisesの欠如」を
    モデルサイズという具体的な変数で示す
```

## ④ Success Criteria（成功基準）

| 指標 | ベースライン | 成功閾値 | 優先度 |
|------|------------|---------|--------|
| CFR(3b NCA) vs CFR(7b NCA) | 14.3%（7b・前回） | 3b > 7b（統計的有意・p<0.05） | 必須 |
| CFR(3b NCA) | 未測定 | > 30% | 必須 |
| CFR(7b NCA・同種) | 未測定 | < CFR(3b NCA) | 必須 |
| 全員一致率(3b) | 未測定 | > 50% | 参考 |
| 全員一致率(7b) | 35%（前回・異種） | 測定のみ | 参考 |

**判定ロジック:**
```
必須指標1: CFR(3b) > CFR(7b)（統計的有意・p<0.05）
  → SUCCESS: モデルサイズがCFRに影響する

必須指標2: CFR(3b NCA) > 30%
  → SUCCESS: 3bではグループシンクが発生している

必須指標3: CFR(7b NCA) < 20%
  → SUCCESS: 7bではグループシンクが抑制されている

3つ全て成立 → FULL SUCCESS
2つ成立     → PARTIAL SUCCESS
1つ以下     → FAIL
```

**各指標の定義（生データから計算）:**
```
CFR = wrong_unanimous / total_unanimous
  wrong_unanimous: 全ノード一致かつ間違いのタスク数
  total_unanimous: 全ノードが同じ答えを出したタスク数

エラー率 = n_wrong / n_total

全員一致率 = total_unanimous / n_total

95% CI: Clopper-Pearson
z検定: CFR(3b) vs CFR(7b)（二比率z検定）
```

## ⑤ Prediction（事前予想）

| 指標 | 予想値 | 根拠 |
|------|--------|------|
| CFR(3b NCA) | 40〜55% | 旧データ（3bモデル）のCFR: 46〜51% |
| CFR(7b NCA・同種） | 10〜20% | 前回（異種7b）: 14.3% |
| 全員一致率(3b) | 70〜90% | 旧データ: 79〜92% |
| 全員一致率(7b・同種) | 30〜50% | 前回（異種7b）: 35% |
| エラー率(3b 単一) | 40〜55% | 旧データ(3b単一): 37〜48% |
| エラー率(7b 単一) | 35〜45% | 前回(7b単一 qwen): 41% |

**予想の確信度:** 中

理由:
  旧データ（3bモデル）が参照値として存在する
  ただし今回はqwen2.5:3bのみで統一
  旧データはqwen2.5:3b / llama3.2:3b / gemma2:2bの混在

**予想が外れた場合に最初に疑うこと:**
```
Case A: CFR(3b) が予想より低い（<30%）
  → 「グループシンクは3bでも発生しない」
  → 旧データ（探索フェーズ）の数字が
    異種モデル混在による影響を受けていた可能性

Case B: CFR(7b 同種) が前回と大きく異なる
  → 「異種モデル vs 同種モデルで
    グループシンクの発生率が変わる」
  → モデルの多様性がCFRに影響する可能性

Case C: 3bと7bのCFRに有意差がない
  → 「グループシンクはモデルサイズに依存しない」
  → 別の変数（タスク難易度・プロンプト設計等）が
    支配的な可能性
```
