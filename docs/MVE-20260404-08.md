```

**接続の論理:**
```

  → 「架空ルール」と「現実常識」で

## ⑤ Prediction（事前予想）

| 指標 | 予想値 | 根拠 |
|------|--------|------|
| CFR(7b同種・常識) | 20〜35% | world_consistency(30.6%)と同程度 |
| CFR(7b異種・常識) | 10〜20% | world_consistency(14.3%)と同程度 |
| CFR差 | +10〜20pp | world_consistencyの+16.3ppと同程度 |
| split率(7b異種) | 40〜70% | world_consistency(65%)と同程度 |
| split_acc(7b異種) | 55〜70% | world_consistency(64.6%)と同程度 |

**予想の確信度:** 中

理由:
  world_consistencyと構造が似ている
  （判断が曖昧・解釈の余地がある）
  ただし「架空ルール」vs「現実常識」で
  モデルのH*_iの分布が異なる可能性がある

**予想が外れた場合に最初に疑うこと:**
```
Case A: CFR差が+5pp未満（FAIL）
  → 「現実常識はモデルが共通して学習している」
  → H*_iのズレが小さい
  → world_consistencyの架空ルールが
    多様性を引き出す本質的な要因

Case B: split率が低い（<20%）
  → 常識推論は7b_hetでも全員一致しやすい
  → 「曖昧さ」が低い
  → タスクの再設計が必要

Case C: CFR差は大きいがsplit_accが低い
  → MVE-05のmath_middleと同じパターン
  → 多様性が「ノイズ」を生む
```

## 常識推論タスクの設計

```
形式（world_consistencyと同じ）:
  Statement: 「文」
  → CORRECT or INCORRECT

world_consistencyとの違い:
  world_consistency:
    World rule: 「この世界では〇〇である」
    Statement: 「〇〇である」
    = 架空のルールを明示する

  常識推論:
    World ruleなし
    Statement: 「現実の常識に基づいた文」
    = 暗黙の常識で判断する

カテゴリと例:

① 自然科学（20問）:
  CORRECT: 「水は0度で凍る（1気圧下）」
  INCORRECT: 「光は音より遅い」
  CORRECT: 「植物は光合成で酸素を生成する」
  INCORRECT: 「鉄は水より軽い」

② 地理・歴史（20問）:
  CORRECT: 「日本は島国である」
  INCORRECT: 「富士山はアルプス山脈の一部である」
  CORRECT: 「東京は日本の首都である」
  INCORRECT: 「太平洋は大西洋より小さい」

③ 日常生活の常識（20問）:
  CORRECT: 「冷蔵庫は食品の腐敗を遅らせる」
  INCORRECT: 「砂糖は塩より重い（同量比較）」
  CORRECT: 「ガラスは透明な素材である」
  INCORRECT: 「電子レンジは金属を安全に加熱できる」

④ 数量・比較（20問）:
  CORRECT: 「1キログラムは1000グラムである」
  INCORRECT: 「1時間は100分である」
  CORRECT: 「哺乳類は鳥類より平均体重が重い」
  INCORRECT: 「地球の直径は月の直径より小さい」

⑤ 因果関係（20問）:
  CORRECT: 「雨が降ると地面が濡れる」
  INCORRECT: 「気温が上がると水が凍りやすくなる」
  CORRECT: 「運動すると心拍数が上がる」
  INCORRECT: 「暗い場所では植物の成長が促進される」

CORRECT/INCORRECT各50問（バランス調整）
```
