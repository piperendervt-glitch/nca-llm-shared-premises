### チェック依頼プロンプト（コピーして使う）

```
以下のMVEシートをレビューしてください。

チェック項目：
1. North Star Linkの妥当性：Claimの成立が本当に北極星への前進になっているか？論理的飛躍はないか？
2. Success Criteriaの妥当性：閾値は意味のある前進を示しているか？測定方法に問題はないか？
3. 予想と結果のズレ分析：差分の考察は妥当か？他に考えられる原因はないか？
4. 次の実験への示唆：この結果から、次に確認すべきことは何か？

判定を以下の形式で出力してください：
- North Star Link: OK / 要修正 / 再設計
- Success Criteria: OK / 要修正 / 再設計
- ズレ分析: 妥当 / 要検討
- 総合: ADVANCE（次の実験へ）/ REVISE（設計修正）/ PIVOT（方向転換）

--- MVEシートここから ---
```
