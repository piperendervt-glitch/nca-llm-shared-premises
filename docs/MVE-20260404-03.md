```
  → 3bモデルの多様性効果は小さい可能性
```

## 実験結果

> 実験完了: 2026-04-04

| 指標 | 予想値 | 実際の値 | 差分 |
|------|--------|---------|------|
| CFR(3b異種) | 15〜25% | 34.0% [20.9%, 49.3%] | 予想より高い |
| 全員一致率(3b異種) | 40〜60% | 47.0% | 予想内 |
| エラー率(3b異種) | 35〜42% | 42.0% | 予想内（上限） |

**達成判定:** PARTIAL（1/2）

```
必須指標1: CFR(3b異種) < CFR(3b同種 38.5%)
  34.0% < 38.5%（-4.5pp）
  → SUCCESS ✅

必須指標2: p < 0.05
  z=0.5233, p=0.6008
  → FAIL ❌
```

**予想とのズレの考察:**
```
CFR(3b異種) = 34.0%（予想: 15〜25%）
  → 予想より高い
  → 7b異種の14.3%のような大幅な低下は起きなかった

全員一致率は大幅低下:
  3b同種: 96% → 3b異種: 47%（-49pp）
  → 異種モデルは意見が分かれやすい
  → しかし、分かれた時の誤り率がまだ高い

多様性効果の全体像:
  3b: 同種38.5% → 異種34.0%（delta: -4.5pp）
  7b: 同種30.6% → 異種14.3%（delta: -16.3pp）
  → 7bの方が多様性の恩恵が大きい
```

**想定外の観察:**
```
エラー率が微増: 39.0% → 42.0%
  → gemma2:2bが2bモデルであることの影響
  → 能力が低いモデルの混在がエラーを増やした可能性

Case Cが該当:
  p=0.60はn=100では検出力不足
  CFRの差は4.5ppと小さい
  
Limitation:
  gemma2:2bは2bモデル（3bクラスではない）
  → モデルサイズの交絡が存在
  → 「多様性の効果」と「能力低下の効果」が混在
  → 真の3bモデルで再実験が必要
```
