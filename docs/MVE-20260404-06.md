## 論理推論タスクの設計

```
形式:
  World rule: 「論理的な前提条件」
  Statement: 「結論の文」
  → CORRECT or INCORRECT

種別と例:

① 三段論法（20問）:
  Rule: 「すべてのAはBである。すべてのBはCである」
  Statement: 「すべてのAはCである」→ CORRECT

② 推移律（20問）:
  Rule: 「XはYより大きい。YはZより大きい」
  Statement: 「XはZより大きい」→ CORRECT
  Statement: 「ZはXより大きい」→ INCORRECT

③ 対偶（20問）:
  Rule: 「AならばBである」
  Statement: 「BでないならばAでない」→ CORRECT
  Statement: 「AでないならばBでない」→ INCORRECT（逆は成立しない）

④ 背理法（20問）:
  Rule: 「AかつBが真であると仮定すると矛盾が生じる」
  Statement: 「AとBは同時に真にはなれない」→ CORRECT

⑤ 複合条件（20問）:
  Rule: 「AかつBならばCである。AはBでない場合Cでない」
  Statement: 「AでありBでもある場合Cである」→ CORRECT
```

## 実験結果

> 実験完了: 2026-04-05

| 指標 | 予想値 | 実際の値 | 差分 |
|------|--------|---------|------|
| CFR(7b同種・論理) | 15〜25% | 3.6% | 予想より大幅に低い |
| CFR(7b異種・論理) | 10〜20% | 4.3% | 予想より低い |
| CFR差 | +5〜15pp | -0.8pp（逆効果） | FAIL |
| split_acc(7b同種・論理) | 60〜80% | 68.8% | 予想内 |
| split_acc(7b異種・論理) | 50〜70% | 51.6% | 予想内 |
| split_acc差 | -10〜+20pp | -17.1pp（逆効果） | FAIL |

**達成判定:** FAIL（0/2）

```
必須指標1: CFR差 > 0
  CFR(7b同種)3.6% vs CFR(7b異種)4.3%
  差: -0.8pp（逆効果）
  → FAIL ❌

必須指標2: split_acc差 > 0
  split_acc(7b同種)68.8% vs split_acc(7b異種)51.6%
  差: -17.1pp（逆効果）
  → FAIL ❌
```

**多様性効果の比較（全タスク）:**
```
タスク         | CFR差    | split_acc差
---------------|----------|------------
world_consist  | +16.3pp  | +17.9pp（有効）
logic_reasoning| -0.8pp   | -17.1pp（逆効果）
math_middle    | -6.6pp   | -46.9pp（逆効果）
```

**タスク種別分析（最重要な発見）:**
```
種別            | homo split_acc | het split_acc | 差分
----------------|---------------|---------------|-------
syllogism       | 100%          | 0%            | -100pp（逆効果・決定論的）
transitivity    | 100%          | 33%           | -67pp（逆効果・決定論的）
contrapositive  | 86%           | 29%           | -57pp（逆効果・決定論的）
contradiction   | 25%           | 73%           | +48pp（有効・解釈の余地あり）
compound        | 50%           | 71%           | +21pp（有効・解釈の余地あり）

→ 答えが形式的に決まるタスク: 多様性が逆効果
  判断に解釈の余地があるタスク: 多様性が有効
```

**予想とのズレの考察:**
```
最大の驚き:
  syllogismとtransitivityで
  7b_homoのsplit_acc=100%
  7b_hetのsplit_acc=0%・33%

  = 決定論的な論理問題では
    同種モデルは意見が分かれた時に
    正解を選べるが
    異種モデルは全く選べない

CFRが予想より低い（3〜4%）:
  7bモデルは論理推論が得意
  全員一致した時の誤りが少ない

split_accの種別差が顕著:
  「判断の曖昧さ」が
  多様性効果の真の決定変数であることを示唆
```

**想定外の観察:**
```
contradiction・compoundで多様性が有効:
  これらは「解釈の余地」がある論理タスク
  world_consistencyと同様に
  多様性が生産的な不一致を生む

= タスクドメイン（知識 vs 論理 vs 計算）ではなく
  「判断の曖昧さ」が多様性効果の決定変数
```

**North Star A-4への接続（精密化）:**
```
旧解釈:
  「知識タスク → 多様性有効」
  「計算タスク → 多様性無効」

新解釈（より精密）:
  「判断が曖昧なタスク → 多様性有効」
    = world_consistency・contradiction・compound
  「答えが決定論的なタスク → 多様性無効」
    = syllogism・transitivity・計算問題

A-4の精密化:
  「shared_premisesの欠如による失敗は
   判断に曖昧さがある場面で最も顕著になる」
  「答えが一意に決まる場面では
   shared_premisesではなく
   能力の問題が支配的」

  = A-4の適用条件が「曖昧さ」という
    より根本的な変数で定義できた
```
